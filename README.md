# Usage

<!-- usage -->

```sh-session

$ hue-cli (--version)
hue-cli/0.0.0 darwin-arm64 node-v21.1.0
$ hue-cli --help [COMMAND]
USAGE
  $ hue-cli COMMAND
...
```
